Typography Mixins

Mixins for controlling font styles.

            
@import 'typography';

.example {
    @include font-size(16px); /* Example usage */
    @include font-weight(700); /* Example usage */
    @include text-color(#333); /* Example usage */
    @include font-size-rem(14px);
}

Animation Mixins

Mixins for adding animations to elements.

            
@import 'animations';

.example {
    @include blink-animation; /* Example usage */
    @include pulse-animation; /* Example usage */
    @include move-animation(1s, 2s); /* Example usage */
    @include rotate-animation(2s); /* Example usage */
    @include background-color-animation(2s); /* Example usage */


    // Define animation name, and properties
    @include keyframes(fade-out) {
      0% { opacity: 1; }
      90% { opacity: 0; }
    }
    // Add animation to element
    .element {
      width: 100px;
      height: 100px;
      background: black;
      @include animation('fade-out 5s 3');
    }
}

Media Queries Mixins

Mixins for creating responsive media queries.

            
.foo {
    @include screen(large) {
        width: 20%;
    }
    @include screen(med) {
        width: 40%;
    }
    @include screen(med-small) {
        width: 60%;
    }
    @include screen(small) {
        width: 80%;
    }
    @include screen(custom, max, 400) {
        width: 100%;
    }
}
